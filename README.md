# Policy Planning

**Full policy planning pipeline from research to proposal.**

Built from 6 real government/campaign projects. Runs axis-based research → spine extraction → proposal → submission cleanup. Includes campaign strategy templates.

### Example Prompts

```
"Draft a cultural policy for the city" → setup→axis research→spine→proposal→cleanup
"Design a campaign strategy" → 8 campaign templates available
```

## Installation

```bash
git clone https://github.com/jasonnamii/policy-planning.git ~/.claude/skills/policy-planning
```

## Update

cd ~/.claude/skills/policy-planning && git pull
